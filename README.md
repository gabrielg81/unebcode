### <div align="center">Aplicação desenvolvida por Gabriel S Cruz 👨‍💻 <br>Mediada pelo prof. Julio Oliveira - Laboratório de aplicações web - UNEB DCET II  🚀</div>  
  

- 🔭 Em Laboratório de Aplicações Web abordaremos os principais conceitos e técnicas para a elaboração de um site ou mesmo sistemas dinâmicos para a internet.  
  

- 🌱 Inicialmente estudaremos HTML e CSS. Essas são as linguagens básicas para o desenvolvimento de páginas web estáticas. Em seguida, trabalharemos tecnologias para criação de páginas dinâmicas que rodam diretamente no servidor, usando NodeJS no backend e ReactiveJs no frontend, ambos usando typescript. Trabalharemos também ferramentas de conexão com banco de dados e padrões de desenvolvimento de software, como MVVM.  
  

- ⚡ Ao final do curso, você será capaz de criar suas próprias páginas na internet e conhecerá os conceitos importantes para a construção de sites web modernos e Dinâmicos.
